## Project Scope

This project demonstrates data-driven testing by reading test cases from a CSV file, comparing expected and actual results, and generating a test report automatically.

## Technologies Used

- Python
- CSV Module

## Features

- Reads test cases from CSV
- Compares expected and actual results
- Displays PASS/FAIL status
- Generates a test report

## How to Run

```bash
python data_driven_testing.py
```

## Output

The program displays test results and creates a `test_report.txt` file.
